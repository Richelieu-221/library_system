# 图书管理系统项目技术介绍
简洁的控制台图书管理示例程序（C语言实现）。功能包括添加/显示/删除图书与借书人、借书与还书。
本项目目前在内存中保存数据（运行时有效），无持久化存储，因此在使用时请保证程序连续运行。

## 文件
- `main.c` - 程序入口，包含主菜单与借还书流程。
- `book.h` / `book.c` - 图书结构与相关操作（添加、显示、查找、删除等）。
- `borrower.h` / `borrower.c` - 借书人结构与相关操作（添加、显示、查找、删除等）。
- 由于本项目借助VS社区版完成项目管理，还包括其他 Visual Studio 工程文件。


## 环境
本项目基于Windows11和Windows10完成了编译和运行测试良好，其中编译环境使用VS(msvc)
- 推荐使用MSVC作为编译环境，也可以使用如gcc等其他编译环境
- 代码使用C语言实现，符合C99语言标准,也兼容C++14及之后的C++标准
- 注意：本代码大量使用了scanf,有缓冲区溢出的风险。在MSVC中禁用了4996内存警告
## 使用说明
运行程序后按菜单提示输入数字选择操作。主要交互示例：
1. 选择 `1` 添加图书（需要输入图书 ID、书名、数量等，根据实现而定）。
2. 选择 `4` 添加借书人（需要输入借书人 ID、姓名等）。
3. 选择 `7` 借书：先输入借书人 ID，再输入图书 ID；若图书存在且库存 >0，则借书成功，借书人记录其 `borrowedBookId`。
4. 选择 `8` 还书：输入借书人 ID，会恢复图书数量并将借书人 `borrowedBookId` 设为 -1。

注意：当前实现每个借书人只能同时借一本书

## 已知限制与扩展建议
- 无数据持久化：可增加文件或数据库存储。
- 借书人只支持借一本书：可改为链表/数组记录多个借阅记录。
